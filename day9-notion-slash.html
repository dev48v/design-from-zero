⌨️ Notion's Slash Menu — Type "/" to Transform a Block

A real document editor with a slash menu

Click into the doc and type / — a command palette pops up. Filter by typing (/head, /todo, /code), use ↑ ↓ to move, Enter to apply. Enter on an empty line makes a new block.

🔨 Build it yourself

    
A document is a list of blocks. The slash menu just rewrites the current block's type. ~70 lines.

    
    
      
  1. 1
    A block = one contenteditable line
    const block = document.createElement("div");
block.contentEditable = true;
block.dataset.type = "text";   // the type is just a string
    2. 
      
  2. 2
    Typing "/" opens the menu
    block.oninput = () => {
  const t = block.textContent;
  if (t.startsWith("/")) openMenu(t.slice(1));   // query after the slash
  else closeMenu();
};
    3. 
      
  3. 3
    Filter commands by the query
    const shown = COMMANDS.filter(c =>
  c.label.toLowerCase().includes(query) ||
  c.keywords.some(k => k.includes(query))
);
    4. 
      
  4. 4
    Arrows move, Enter applies
    if (menuOpen) {
  if (e.key === "ArrowDown") sel = (sel+1) % shown.length;
  if (e.key === "ArrowUp")   sel = (sel-1+shown.length) % shown.length;
  if (e.key === "Enter")     apply(shown[sel]);   // and preventDefault
}
    5. 
      
  5. 5
    Apply = change the type, clear the slash
    function apply(cmd){
  block.dataset.type = cmd.type;     // restyles via CSS
  block.textContent = "";            // wipe "/head"
  closeMenu(); placeCaret(block);
}
