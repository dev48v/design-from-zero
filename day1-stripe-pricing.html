🔨 Build it yourself

    
Estimated time: 15 minutes. Skills needed: zero. Just copy + paste.


    
    
      
      
  1. 
        
    
          
    1
    
          
    Make an empty folder + file
    
        
    
        
    On your laptop, anywhere:
    
        
    

    mkdir my-pricing
cd my-pricing
touch index.html
    
          
        
    
        
    On Windows: right-click in folder → New → Text file → rename to index.html
    
      
    2. 

      
      
  2. 
        
    
          
    2
    
          
    Paste this in index.html
    
        
    
        
    The whole project is one HTML file. No build, no npm, no terminal.
    
        
    

    <!DOCTYPE html>
<html>
<head>
  <script src="https://cdn.tailwindcss.com"></script>
</head>
<body class="bg-slate-50 min-h-screen flex items-center justify-center p-8">
  <div class="max-w-5xl w-full">
    <h1 class="text-4xl font-bold text-center mb-2">Pricing</h1>
    <p class="text-center text-slate-500 mb-12">Start free. Scale as you grow.</p>

    <div class="grid md:grid-cols-3 gap-6">
      <!-- Free -->
      <div class="bg-white rounded-2xl p-8 border border-slate-200 hover:-translate-y-1 hover:shadow-2xl transition">
        <div class="text-sm font-semibold text-indigo-600 uppercase">Free</div>
        <div class="mt-4 text-5xl font-bold">$0</div>
        <ul class="mt-8 space-y-3 text-sm text-slate-600">
          <li>✓ 100 API calls/day</li>
          <li>✓ Community support</li>
        </ul>
        <button class="mt-8 w-full py-3 rounded-lg border border-slate-300 font-medium">Get started</button>
      </div>

      <!-- Pro (highlighted) -->
      <div class="bg-gradient-to-br from-indigo-600 to-purple-600 rounded-2xl p-8 text-white relative">
        <div class="absolute top-4 right-4 bg-white/20 text-xs font-semibold px-3 py-1 rounded-full">POPULAR</div>
        <div class="text-sm font-semibold uppercase">Pro</div>
        <div class="mt-4 text-5xl font-bold">$29</div>
        <ul class="mt-8 space-y-3 text-sm text-indigo-100">
          <li>✓ Unlimited API calls</li>
          <li>✓ Priority support</li>
        </ul>
        <button class="mt-8 w-full py-3 rounded-lg bg-white text-indigo-600 font-semibold">Start free trial</button>
      </div>

      <!-- Enterprise -->
      <div class="bg-white rounded-2xl p-8 border border-slate-200">
        <div class="text-sm font-semibold text-indigo-600 uppercase">Enterprise</div>
