🔨 Build it yourself

    
Four columns, two flex tricks, one grid row. Start with the shell and fill in the pieces.

    
    
      
  1. 
        
    1
    The outer shell: four flex children
    
        
    .dc-shell {
  display: flex;        /* all four panels side by side */
  height: 520px;
  overflow: hidden;
}
/* Each panel has a fixed width except .dc-main which gets flex:1 */
    
      
    2. 
      
  2. 
        
    2
    Server rail — pill-to-squircle on hover
    
        
    .dc-icon {
  width: 48px; height: 48px;
  border-radius: 50%;           /* pill by default */
  transition: border-radius .15s;
}
.dc-icon:hover { border-radius: 16px; } /* squircle on hover */

/* Active indicator: pseudo-element on the left */
.dc-icon::before {
  content: '';
  position: absolute; left: -8px;
  width: 4px; background: #fff;
  border-radius: 0 4px 4px 0;
  transition: height .15s;
  height: 0;
}
.dc-icon.active::before { height: 40px; }
    
      
    3. 
      
  3. 
        
    3
    Channel list typography — categories + items
    
        
    .dc-category {
  font-size: 11px; font-weight: 700;
  text-transform: uppercase;
  letter-spacing: .04em;
  color: #96989d;
}
.dc-ch-item {
  display: flex; align-items: center;
  gap: 6px; padding: 5px 8px 5px 16px;
  border-radius: 4px; color: #96989d;
}
.dc-ch-item:hover  { background: #35363c; color: #d5d7dc; }
.dc-ch-item.active { background: #404249; color: #f2f3f5; }
    
      
    4. 
      
  4. 
        
    4
    Message rows — CSS grid not flex
    
        
    .dc-msg-group {
  display: grid;
  grid-template-columns: 40px 1fr; /* avatar | content */
  gap: 0 12px;
}
/* Avatar spans rows 1 and 2; username+text each sit in col 2 */
.dc-msg-av  { grid-column: 1; grid-row: 1 / 3; }
.dc-msg-meta { grid-column: 2; }    /* username + timestamp */
.dc-msg-text { grid-column: 2; }    /* the actual text */
    
      
    5. 
      
  5. 
        
    5
    The Discord color palette — blurple + dark grays
    
        
    /* Core palette */
--dc-bg-1: #313338;   /* main chat bg */
--dc-bg-2: #2b2d31;   /* sidebar bg */
--dc-bg-3: #1e1f22;   /* server rail bg */
--dc-blurple: #5865f2; /* brand accent */
--dc-green:   #3ba55c;
--dc-red:     #ed4245;

/* Text hierarchy */
--dc-text-hi: #f2f3f5;   /* primary */
--dc-text-md: #dcddde;   /* message body */
--dc-text-lo: #96989d;   /* muted / channels */
    
      
    6. 
      
  6. 
        
    6
    Status dots — CSS pseudo-element
    
        
    /* One trick for all 4 statuses */
.dc-member-av::after {
  content: '';
  position: absolute; bottom: 0; right: 0;
  width: 10px; height: 10px;
  border-radius: 50%;
  border: 2px solid #2b2d31; /* "cut out" trick */
}
.online::after { background: #3ba55c; }
.idle::after   { background: #faa81a; }
.busy::after   { background: #ed4245; }
.offline::after{ background: #747f8d; }
