🍎 macOS Dock — Magnify on Hover

The Dock from macOS — the fisheye magnify

Move your mouse along the dock below. The icon under the cursor grows largest, its neighbours grow a little, and everything else stays small — a smooth fisheye lens. Click an icon to make it bounce.

The math: for each icon we measure the horizontal distance d from its centre to the cursor. Scale falls off with distance: scale = 1 + maxBoost · max(0, 1 − d/range). Under the cursor d=0 → full boost; past range px → back to 1×.

  3. 3
    Distance → scale, the falloff function
    
        
    const MAX = 0.9;   // biggest extra scale (1.9× under the cursor)
const RANGE = 110; // px of influence on each side

function scaleFor(distance) {
  const t = Math.max(0, 1 - Math.abs(distance) / RANGE);
  return 1 + MAX * t;   // d=0 → 1.9× ; d≥RANGE → 1×
}
    
      
    4. 

      
  4. 4
    On mousemove, scale every icon
    
        
    const items = [...dock.querySelectorAll(".dock-item")];

dock.addEventListener("mousemove", (e) => {
  items.forEach(item => {
    const box = item.getBoundingClientRect();
    const center = box.left + box.width / 2;
    item.style.transform = `scale(${scaleFor(e.clientX - center)})`;
  });
});
    
      
    5. 

      
  5. 5
    Reset when the mouse leaves
    
        
    dock.addEventListener("mouseleave", () => {
  items.forEach(item => item.style.transform = "scale(1)");
});
/* the CSS transition makes the shrink-back smooth too */

  6. 6
    Bounce on click (keyframe)
    
        
    @keyframes dockBounce {
  0%,100% { transform:translateY(0); }
  30%     { transform:translateY(-26px); }
  72%     { transform:translateY(-8px); }
}
item.addEventListener("click", () => {
  item.classList.add("bounce");
  item.addEventListener("animationend",
    () => item.classList.remove("bounce"), { once:true });
});
